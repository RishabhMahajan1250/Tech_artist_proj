# Tech_artist_proj
Given task: - to visually enhance this prototype and transform it into a polished, engaging experience similar in feel to games like Dashy Crashy.

My Approach : 
Art Direction : 
      Theme - To incorporate a warm, well-lit, and high-contrast, punchy setting, I decided to build upon a dessert theme with a suitable color palette. 
      Stylized SkyBox - Created a custom shader for skybox with animated clouds, sun, and tweakable related properties.
      Background Props - Added simple low-poly environment assets ( Pyramids, Cactus)
